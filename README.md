# 🎓 Student Management System

A desktop-based **Student Management System** developed using **Java Swing**, **JDBC**, and **MySQL**. This application allows users to manage student records through a graphical user interface.

## 📌 Features

- ➕ Add new students
- 📋 View all students
- ✏️ Update student details
- ❌ Delete student records
- 🔍 Search students by ID
- 🖥️ User-friendly Java Swing interface
- 💾 MySQL database connectivity using JDBC

## 🛠️ Technologies Used

- Java
- Java Swing
- JDBC (Java Database Connectivity)
- MySQL
- Git
- GitHub


## 🚀 How to Run

1. Clone this repository.
2. Open the project in Visual Studio Code or Eclipse.
3. Add the MySQL JDBC Connector to the project.
4. Create the required MySQL database and `students` table.
5. Update the database credentials in `DBConnection.java`.
6. Run `StudentManagementGUI.java` (or `Main.java`) to start the application.
